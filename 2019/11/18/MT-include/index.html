MT-include

Author: LGF Date: November 18, 2019  14:06:16

可以读取外部的 module 和文件，
注意：在定义了 MT-include 之后，即使被引用的文件被更新了，该页面也不能自动更新（需要再构筑），如果需要其自动更新的话，请使用动态引用 SSI 或者 PHP Include。

用法

代码会被编译成

常用修饰符

file=”file_foo”

引用指定文件，文件里的内容可以被展开
例：

module=”module_foo”

引用 MT module，module 内的内容可以被展开，而且 module 会读取被展开场所的上下文环境。

parent=”0 | 1”

当设定 parent 为 1 时，会自动调取 website 级别的 module (如果当前环境为 blog)，如果 website 里没有对应名称的 module 则什么都不做。
